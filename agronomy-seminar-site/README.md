# Química aplicada à Engenharia Agronômica — Site do Seminário

**Projeto**
Site acadêmico para apoio visual ao seminário “Química aplicada à Engenharia Agronômica”, com linguagem didática, navegação clara e componentes reutilizáveis.

**Estrutura**
- index.html
- pages/introducao.html
- pages/solo-fertilizantes.html
- pages/defensivos-agua.html
- pages/sustentabilidade-tecnologia.html
- pages/desafio-agronomo.html
- assets/css/styles.css
- assets/js/script.js
- components/header.html
- components/footer.html
- components/card.html

**Como executar**
1. Abra o arquivo `index.html` no navegador.

**Créditos**
Conteúdo baseado no texto acadêmico “Química aplicada à Engenharia Agronômica: solo, insumos e sustentabilidade” e em referências listadas no rodapé das páginas.
